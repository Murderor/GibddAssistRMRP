# ГИБДД Helper — RMRP

Приложение для просмотра нормативно-правовых актов (КоАП, УК РФ, других кодексов) с функцией инкриминирования статей, автоматическим расчётом наказания и удобным копированием списка для вставки в игровой или текстовый документ.



---

## 📌 Возможности

- 📚 **Просмотр статей** из нескольких кодексов (разделы, главы, диспозиция, санкция).  
- 🔍 **Поиск** по номеру статьи, диспозиции, санкции, кодексу, разделу, главе.  
- ⌨️ **Горячие клавиши**:  
  - Автоматический фокус на строке поиска при активации окна.  
  - Клавиша **Enter** в списке статей для быстрого инкриминирования.  
- ⚖️ **Инкриминирование статей** с выбором конкретного вида наказания, если санкция предусматривает альтернативы (например, «штраф **или** лишение прав»).  
- 💰 **Ввод суммы штрафа** для диапазонов (например, «от 20 000 до 50 000 рублей») через диалоговое окно с числовым полем.  
- 📊 **Автоматический расчёт** общей суммы штрафов и признака лишения прав.  
- 📋 **Копирование в буфер обмена** списка инкриминированных статей в формате `12.8 КоАП, 12.26 КоАП, 264.1 УК РФ`.  
- 🗑️ **Удаление отдельных статей** из списка инкриминированных через контекстное меню (правая кнопка мыши).  
- 🖱️ **Изменяемые размеры колонок** – перетаскивайте границы между панелями мышью.  
- 🎨 **Современный тёмный интерфейс** на PyQt6.  
- 📁 **Гибкая загрузка данных** – программа использует внешнюю папку `NPA`, расположенную рядом с исполняемым файлом.  
- ❓ **Инструкция при первом запуске** – если папка `NPA` отсутствует или в ней нет JSON-файлов, отображается подробное руководство по добавлению кодексов.

---

## ⚙️ Установка и запуск

1. Скачайте последнюю версию `GibddAssist.exe` со [страницы релизов](https://github.com/Murderor/GIBDDAssist/releases/latest).  
2. Поместите `.exe` в любую удобную папку (не требуется установка).  
3. **Важно:** создайте рядом с `.exe` папку `NPA` и поместите в неё JSON-файлы с кодексами (см. раздел «Формат JSON»).  
4. При первом запуске, если папка `NPA` отсутствует, появится диалог с инструкцией.

## 📄 Формат JSON-файлов

Приложение загружает все `.json` файлы из папки `NPA`. Ожидается следующая структура:

```json
{
  "codex_name": "КоАП",
  "sections": [
    {
      "section_name": "РАЗДЕЛ II. ОСОБЕННАЯ ЧАСТЬ",
      "chapters": [
        {
          "chapter_name": "Глава 9. Правонарушения в области дорожного движения",
          "articles": [
            {
              "article_number": "12.8",
              "disposition": "Управление транспортным средством в состоянии опьянения...",
              "penalty": "штраф 30000 рублей или лишение прав до 2 лет",
              "fine_amount": 30000
            }
          ]
        }
      ]
    }
  ]
}
```

**Пояснения полей:**

- `codex_name` – название кодекса (отображается в интерфейсе).  
- `sections` / `chapters` – необязательные, но улучшают навигацию.  
- `articles` – массив статей:  
  - `article_number` – номер статьи (может содержать точки, например «12.8.1»).  
  - `disposition` – описание правонарушения.  
  - `penalty` – текст санкции (поддерживаются альтернативы через **« или »**, диапазоны **«от X до Y рублей»**).  
  - `fine_amount` – **необязательное поле**. Если указано, используется как базовая сумма (минимальная при диапазоне). Если отсутствует или равно 0, программа попытается извлечь сумму из текста `penalty`.  

**Числа** могут содержать пробелы или точки в качестве разделителей тысяч (например, `20.000` или `50 000`).

## 🖥️ Использование

1. **Поиск статей** – при активации окна фокус автоматически устанавливается на строку поиска, можно сразу начинать печатать. Введите текст (номер статьи, ключевое слово, название кодекса и т.д.).  
2. **Просмотр статьи** – кликните на статью в левом списке или перемещайтесь по списку клавишами стрелок; в правой панели отобразятся диспозиция и санкция.  
3. **Инкриминирование**:  
   - Нажмите кнопку **«⚖️ Инкриминировать статью»** **ИЛИ** просто нажмите **Enter**, когда статья выделена в списке.  
   - Если санкция содержит **несколько альтернатив** (разделённых «или»), откроется диалог выбора конкретного наказания.  
   - Если выбран штраф с **диапазоном** (например, «от 20 000 до 50 000 рублей»), появится окно для ввода конкретной суммы в указанных пределах.  
   - Если наказание единственное (фиксированный штраф или лишение прав), оно применяется автоматически.  
4. **Список инкриминированных статей** отображается в левой панели. Там же показана общая сумма штрафа и наличие лишения прав.  
5. **Удаление статьи** – правый клик на статье в списке инкриминированных → пункт «❌ Удалить статью».  
6. **Изменение размеров** – перетаскивайте границу между панелями для удобного просмотра.  
7. **Текст для вставки** (поле «Текст для вставки в игру») содержит только перечень инкриминированных статей с указанием кодекса, например:  
   `12.8 КоАП, 12.26 КоАП, 264.1 УК РФ`  
   Нажмите **«📋 Копировать в буфер обмена»**, чтобы скопировать этот текст.  
8. **Очистить список** – кнопка **«🗑️ Очистить все»** удаляет все инкриминированные статьи (с подтверждением).
